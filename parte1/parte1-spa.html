SPA frameworks

Um SPA é uma aplicação web que roda em uma única página, oferecendo a mesma experiência de uso de um aplicativo desktop. Um bom exemplo é o Gmail do Google, a navegação na aplicação acontece toda em uma única página e todo o conteúdo é carregado de uma vez ou obtido de forma assíncrona conforme necessário.

  • Redução de requisições ao servidor
  • Renderização da view inteiramente no browser cliente
  • roteamento e gerenciamento de estado no browser cliente
  • Caching: LocalStorage e SessionStorage
  • Mais uma peça importante para obter escalabilidade

TypeScript

Superconjunto de JavaScript desenvolvido pela Microsoft que adiciona tipagem e alguns outros recursos a linguagem. Pode ser usada para desenvolver aplicações JavaScript no lado cliente e lado servidor (Node.js). Anders Hejlsberg, arquiteto da linguagem C# e criador das linguagens Delphi e Turbo Pascal, trabalhou no desenvolvimento do TypeScript.

  • Tipagem estática (opcional): variáveis, parâmetros, retornos de função
  • Checagem de tipo em tempo de compilação
  • Inferência de tipos
  • Interfaces
  • Enums
  • Tipos genéricos
  • Namespaces
  • Tuplas
  • Classes
  • Modules
  • parâmetros opcionais e default
Prática

> mkdir pratica-typescript

> cd pratica-typescript

> npm init -y

> npm install -D typescript

> npx tsc -w index.ts
