Node.js

Node.js é um interpretador open source de código JavaScript de modo assíncrono e orientado a eventos, focado em migrar a programação do Javascript do lado do cliente para os servidores, criando assim aplicações de alta escalabilidade, capazes de manipular milhares de conexões/requisições simultâneas em tempo real, numa única máquina física.

O Node.js é baseado no interpretador V8 JavaScript Engine (interpretador de JavaScript open source implementado pelo Google em C++ e utilizado pelo Chrome)

  • Javascript no cliente e no servidor
  • Criação de aplicações distribuídas com diferentes protocolos (socket, udp, http, http2 ...)
  • Rico repositório de pacotes reutilizáveis (NPM)
  • Otimizações que equiparam sua velocidade a de linguagens já consagradas
  • Single-threaded, orientado a eventos, callbacks para operações de I/O - atinge um nível de escalabilidade que o permite gerenciar eficientemente centenas de milhares de conexões simultâneas.
  • Melhor cenários: aplicações que fazem bastante uso de I/O
  • Ñ recomendado: aplicações que fazem uso significativo da CPU
Npm

Npm é um gerenciador de pacotes para a linguagem JavaScript. É o gerenciador de pacotes padrão para o Node.js. Contém um executável de linha de comando (npm) que acessa o site https://www.npmjs.com/ que é um repositório online de pacotes publicos e privados.
