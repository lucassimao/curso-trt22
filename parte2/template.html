Desenvolvimento WEB com Angular e SpringBoot / Spring framework

Spring Framework

É o framework open source mais popular para aplicações Java criado como resposta a complexidade envolvida na criação de aplicações corporativas durante as primeiras versões do J2EE.

Sub-projetos

Pode ser visto como um conjunto de módulos(sub-projetos) para os mais variados serviços, dentre os mais relevantes:

  • Spring Core Container
  • Programação orientada a aspectos
  • Autenticação e autorização - via Spring Security)
  • Acesso a dados de bancos relacionais (JDBC) e não relacionais
  • Container de inversão de controle e injeção de dependência
  • Messageria via Java Message Service (JMS)
  • Spring MVC: Uma abstração das apis de Servlet
  • Gerenciamento de transação
  • Gerenciamento remoto - abstrai JMX
  • Teste - módulo com classes utilitárias para escrever testes unitários e de integração
Spring MVC

Framework para desenvolvimento de aplicações WEB, baseado no modelo de requisição e resposta HTTP

Prática

> mvn archetype:generate -DgroupId=br.jus.trt22 -DartifactId=pratica-spring-mvc -DarchetypeArtifactId=maven-archetype-quickstart

> cd pratica-spring-mvc

  • pom.xml
  • src/main/java/br/jus/trt22/WebConfig.java
  • src/main/java/br/jus/trt22/JpaConfig.java
  • src/main/java/br/jus/trt22/App.java
  • src/main/java/br/jus/trt22/Advogado.java
  • src/main/java/br/jus/trt22/AdvogadoRepository.java
  • src/main/java/br/jus/trt22/AdvogadoController.java

> mvn clean package

> cp target/pratica-spring-mvc-1.0-SNAPSHOT.war apache-tomcat-9.0.22/webapps/ROOT.war

> ./apache-tomcat-9.0.22/bin/startup.sh
